# 🧪 Testes Automatizados – Desafio Técnico QA

Este repositório contém a implementação de testes automatizados desenvolvidos como parte de um desafio técnico de QA.

O objetivo foi:

- Identificar e reportar bugs nas páginas avaliadas
- Especificar o comportamento esperado utilizando BDD
- Implementar testes automatizados com Cypress para validar funcionalidades

---

## 📌 Páginas testadas

- https://qualidade.apprbs.com.br/site  
- https://qualidade.apprbs.com.br/certificacao  

---

## 🐞 Escopo de Validação

Os testes automatizados cobrem:

- Fluxo de inscrição na newsletter
- Fluxo de inscrição para certificação
- Validação de redirecionamentos externos
- Validação de links do header
- Comportamento de botões institucionais

Os cenários foram estruturados com base no comportamento esperado da aplicação.

---

## 🚀 Tecnologias Utilizadas

- Node.js
- Cypress
- JavaScript
- Gherkin (estruturação de cenários)
