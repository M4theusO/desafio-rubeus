Item 01 - Erro ao tentar se inscrever
Tipo: Correção
Classificação: Utilidade
Prioridade: Alta
Descrição: Após acessar a página "https://qualidade.apprbs.com.br/certificacao", informar os dados necessários para se inscrever e clicar em avançar, surge uma mensagem de erro dizendo "É necessário informar a base legal". Impossibilitando continuar o fluxo.

Item 02 - Ícone do YouTube redireciona para TikTok
Tipo: Correção
Classificação: Utilidade
Prioridade: Média
Descrição: Após acessar a página "https://qualidade.apprbs.com.br/certificacao", rolar a tela até o final e clicar no ícone do Youtube, o usuário é redirecionado para o Tiktok.

Item 03 - Botão "Quero me certificar" redireciona para o Google
Tipo: Correção
Classificação: Utilidade
Prioridade: Média
Descrição: Após acessar a página "https://qualidade.apprbs.com.br/certificacao", rolar a tela até o final e clicar no último botão "Quero me certificar", o usuário é redirecionado para o site de pesquisa Google. Os outros botões estão redirecionando para "https://rubeus.com.br/".

Item 04 - Ao clicar no botão "Saiba mais" nada acontece
Tipo: Correção
Classificação: Utilidade
Prioridade: Média
Descrição: Após acessar a página https://qualidade.apprbs.com.br/certificacao e clicar no botão "Saiba mais" nada acontece. O mesmo acontece com os botões "Saiba mais" nos cards na sessão "Outros Cursos".
