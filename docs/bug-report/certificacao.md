## 🔴 Item 01 – Erro ao tentar se inscrever

**Tipo:** Correção  
**Classificação:** Utilidade  
**Prioridade:** Alta  

### Passos para reproduzir
1. Acessar a página https://qualidade.apprbs.com.br/certificacao  
2. Preencher os campos obrigatórios do formulário de inscrição 
3. Clicar em **Avançar**

### Resultado esperado
O sistema deve permitir a continuidade do processo de inscrição.

### Resultado atual
É exibida a mensagem:  
> "É necessário informar a base legal"

O fluxo é interrompido.

## 🟠 Item 02 – Ícone do YouTube redireciona para TikTok

**Tipo:** Correção  
**Classificação:** Usabilidade  
**Prioridade:** Média  

### Passos para reproduzir
1. Acessar a página https://qualidade.apprbs.com.br/certificacao  
2. Rolar até o rodapé  
3. Clicar no ícone do **YouTube**

**Tipo:** Correção  
**Classificação:** Usabilidade  
**Prioridade:** Média
