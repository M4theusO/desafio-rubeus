# 🐞 Relatório de Bugs – Página de Certificação

**URL testada:**  
https://qualidade.apprbs.com.br/certificacao

---

## 🔴 Item 01 – Erro ao tentar se inscrever

**Tipo:** Correção  
**Classificação:** Utilidade  
**Prioridade:** Alta  

**Descrição:**  
Após acessar a página, informar os dados necessários para se inscrever e clicar em **Avançar**, surge a mensagem:

> "É necessário informar a base legal"

O fluxo é interrompido, impossibilitando continuar a inscrição.

---

## 🟠 Item 02 – Ícone do YouTube redireciona para TikTok

**Tipo:** Correção  
**Classificação:** Utilidade  
**Prioridade:** Média  

**Descrição:**  
Ao acessar a página, rolar até o final e clicar no ícone do **YouTube**, o usuário é redirecionado para o TikTok.

---

## 🟠 Item 03 – Botão "Quero me certificar" redireciona para o Google

**Tipo:** Correção  
**Classificação:** Utilidade  
**Prioridade:** Média  

**Descrição:**  
Ao rolar até o final da página e clicar no último botão **"Quero me certificar"**, o usuário é redirecionado para o Google.

Os demais botões estão redirecionando corretamente para:  
https://rubeus.com.br/

## 🟠 Item 04 – Botão "Saiba mais" não executa ação

**Tipo:** Correção  
**Classificação:** Utilidade  
**Prioridade:** Média  

**Descrição:**  
Ao clicar no botão **"Saiba mais"**, nenhuma ação é executada.

O mesmo comportamento ocorre com os botões "Saiba mais" na seção **Outros Cursos**.
